# finance-test
program keuangan sederhana
WebFinance adalah aplikasi web sederhana yang dirancang untuk membantu pengguna dalam mencatat pemasukan dan pengeluaran keuangan secara sementara tanpa menggunakan database.
Aplikasi ini dikembangkan menggunakan HTML5, CSS3, dan JavaScript dengan pendekatan Object Oriented Programming (OOP), serta menerapkan pemisahan antar file (modular programming).

Proyek ini digunakan sebagai media pembelajaran bagi siswa kelas XI Rekayasa Perangkat Lunak (RPL) SMK Pusdikhubad untuk memahami implementasi logika OOP pada pemrograman web menggunakan JavaScript murni (Vanilla JS), termasuk pengelolaan data sementara (in-memory) dan ekspor/impor data ke file Excel menggunakan library SheetJS (XLSX).
